# java-baseball-precourse

## 프로젝트 개요

"숫자 야구" 프로젝트는 컴퓨터가 1부터 9까지의 서로 다른 숫자 3개를 임의로 선택하면, 사용자가 3자리 숫자를 입력하며 힌트를 바탕으로 정답을 맞추는 콘솔 게임입니다.
사용자가 입력한 숫자와 컴퓨터의 숫자를 비교하여 스트라이크 / 볼 / 낫싱(포볼) 결과를 출력하고, 3스트라이크가 되면 게임이 종료됩니다.
게임 종료 후에는 재시작(1) 또는 완전 종료(2) 를 선택할 수 있습니다.
또한 사용자가 잘못된 값을 입력하면 [ERROR]로 시작하는 에러 메시지를 출력하고 게임은 계속 진행되어야 합니다.

## 주요 기능

1. 컴퓨터의 3자리 정답(1~9, 중복 없음) 생성 기능
2. 사용자 입력(3자리 숫자) 기능 및 입력값 검증 기능
3. 스트라이크/볼/낫싱 판정 기능
4. 게임 진행(정답 맞출 때까지 반복) 기능
5. 게임 종료 후 재시작/종료 선택 기능
6. 잘못된 입력 시 에러 메시지 출력 후 게임 유지 기능

## 구현 기능 목록

### 1. 입력 기능
- [x] : **사용자에게 3자리 숫자를 입력 받는 기능**
    - 입력이 숫자가 아닌 경우 예외 처리
    - 입력 길이가 3이 아닌 경우 예외 처리
    - 각 자리가 1~9 범위를 벗어나는 경우 예외 처리 
    - 중복된 숫자가 포함된 경우 예외 처리 
- [x] : **게임 종료 후 재시작/종료 입력을 받는 기능**
    - 입력값이 1 또는 2가 아닌 경우 예외 처리 

### 2. 비즈니스 로직 처리 기능
- [x] : **컴퓨터의 정답 숫자 3개를 생성하는 기능**
    - 1~9 범위에서 중복 없이 3개 선택 
    - 3자리로 관리(예: [4, 2, 5]) 
- [x] : **입력값과 정답을 비교하여 스트라이크/볼 개수를 계산하는 기능**
    - 같은 숫자 & 같은 위치: 스트라이크 
    - 같은 숫자 & 다른 위치: 볼 
    - 공통 숫자 없음: 낫싱(또는 포볼) 
- [x] : **게임 종료 조건을 판단하는 기능**
    - 스트라이크가 3개이면 게임 종료 
- [x] : **게임 재시작 시 정답을 새로 생성하고 상태를 초기화하는 기능**

### 3. 출력 기능
- [x] : **입력 안내 메시지를 출력하는 기능**
    - “숫자를 입력해주세요 : ” 형태로 입력 유도 
- [x] : **판정 결과(스트라이크/볼/낫싱)를 출력하는 기능**
    - 예) 1스트라이크, 1스트라이크 1볼, 낫싱 
- [x] : **정답을 맞혔을 때 종료 메시지를 출력하는 기능**
    - 예) 3스트라이크 
    - 예) 3개의 숫자를 모두 맞히셨습니다! 게임 종료 
- [x] : **재시작/종료 안내 메시지를 출력하는 기능**
    - 예) 게임을 새로 시작하려면 1, 종료하려면 2를 입력하세요.
- [x] : **잘못된 입력 시 에러 메시지를 출력하는 기능**
    - 모든 에러 메시지는 [ERROR]로 시작 
    - 에러 발생 후에도 게임은 종료되지 않고 계속 진행되어야 함
